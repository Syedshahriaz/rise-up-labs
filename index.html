This 12 months Professional Diploma in Game Development program is for all gaming enthusiasts who wants to build a career in exponentially growing gaming industry. Total training duration combine with 500 hours in house course and 4 months of industrial training.

Objective

  • To introduce the game industry
  • To develop 2D & 3D game development using game engine
  • To manage version of game using version control system
  • To create game asset using 2D & 3D asset creation tools
  • To discuss and define the terms & principles of game design
  • To provide soft skill training to adapt with teamwork & improve communication skill
  • To enable trainee find bugs and fix them
  • To provide knowledge of monetization strategies & integrate common monetization strategies.
  • To enable trainee to make their own game
  • To guide trainee about different career path through game development
